    Risk attitudes and their characterizations.

    Main definitions #

    • IsAffineUtility — utility function of the form u(x) = a·x + b
    • IsRiskNeutral — utility preserves expected values
    • IsLinearUtility — utility linear over lottery mixtures

    def IsAffineUtility {𝕜 : Type u_1} [Field 𝕜] (u : 𝕜 → 𝕜) :
    Prop

    A utility function is affine (linear + constant): u(x) = a·x + b. An agent with an affine utility function is risk neutral. [MSZ 2.24]

    Equations
    • IsAffineUtility u = ∃ (a : 𝕜) (b : 𝕜), ∀ (x : 𝕜), u x = a * x + b

      def IsRiskNeutral {𝕜 : Type u_1} [Field 𝕜] [LinearOrder 𝕜] {I : Type u_2} [Fintype I] (u : 𝕜 → 𝕜) :
      Prop

      Risk neutrality for lotteries over a finite index set I: u(∑ pᵢ·xᵢ) = ∑ pᵢ·u(xᵢ). Equivalent to u being affine. [MSZ 2.27]
