    EconCSLib.SocialChoice.FairDivision.Divisible.Instance #

    Bundled semantic interfaces for divisible-goods fair division.

    This file sits above the raw divisible allocation layer. It keeps Divisible.Allocation N Ω := FairDivision.Allocation N (Set Ω) and Divisible.IsAllocation as the low-level feasibility vocabulary, while exposing canonical bundled instance types for ordinal, cardinal, and measure-based divisible-goods problems.
