    EconCSLib.SocialChoice.FairDivision.Basic #

    Generic fair-division vocabulary as a structured special case of social choice.

    This file introduces:

    • Allocation N S as the common shape of an allocation
    • Instance for fully general fair-division problems with preferences over complete allocations
    • ShareInstance for the standard no-externality model where each agent ranks only the share they receive
    • the lift ShareInstance.toInstance, showing how fair division fits into the generic social-choice layer
